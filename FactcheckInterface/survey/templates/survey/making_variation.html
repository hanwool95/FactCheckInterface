{% elif task == 1 %} Task 유형 2. 부정문으로 만들기: 해당 클레임과 반대 의미의 문장 만들기
단순히 ‘~하지 않는다’로 변형하는 것이 아니라, 해당 문장이 참일 때 거짓일 문장을 만드는 것을 권장한다. 단어 자체를 바꾸거나, 수량을 바꾸는 것을 통해 부정문으로 만들 수 있다. 표제어는 그대로 사용한다.
예시)
대한민국에서 재배한 담배는 모두 한국담배인삼공사가 구매한다.
=> (수량 변경)대한민국에서 재배한 담배의 약 80%만 한국담배인삼공사가 구매한다.
=> (단어 변경)대한민국에서 재배한 담배는 모두 한국담배인삼공사가 재배한다.
{% elif task == 2 %} Task 유형 3. 문장의 일부를 다른 것으로 대체하기
클레임의 표제어는 유지한 채 동사나 목적어를 변형한다. 같은 종류의 단어(나라 이름, 식물 종류 등)와 다른 종류의 단어를 모두 대체어로 사용할 수 있다. 만들어진 문장과 원본 문장은 관련성이 없어야 한다.
예시)
대한민국에서 재배한 담배는 모두 한국담배인삼공사가 구매한다.
=> (목적어 변형: 같은 종류의 단어, 작물)대한민국에서 재배한 인삼은 모두 한국담배인삼공사가 구매한다.
=> (동사 변형)대한민국에서 재배한 담배는 모두 한국담배인삼공사가 판매한다.
=> (다른 종류의 단어)대한민국의 담배 생산 공장은 모두 한국담배인삼공사 소유이다.
{% elif task == 3 %} Task 유형 4. 기존의 문장보다 더 구체적인 수준의 문장 만들기

예시)

=> (충청도 ∈ 대한민국)충청도에서 재배한 담배는 모두 한국담배인삼공사가 구매한다.

예시)
